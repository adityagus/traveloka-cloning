<h2>Overview</h2>
<p>Unlock the potential of your travel booking projects with <strong>Traveloka-cloning</strong>, a robust framework built on Next.js and Tailwind CSS.</p>
<p><strong>Why Traveloka-cloning?</strong></p>
<p>This project simplifies the development of dynamic travel applications while enhancing user experience. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🌐 <strong>Next.js Integration:</strong> Simplifies server setup and routing for dynamic content delivery.</li>
<li class="my-0">🎨 <strong>Tailwind CSS Support:</strong> Enables utility-first styling for responsive design, enhancing UI development.</li>
<li class="my-0">🔍 <strong>ESLint Configuration:</strong> Ensures code quality and maintainability with React-specific linting rules.</li>
<li class="my-0">⚡ <strong>Real-time Development:</strong> Facilitates local development with live updates, improving workflow efficiency.</li>
<li class="my-0">🧩 <strong>Modular Architecture:</strong> Promotes reusability and organization of components, making collaboration easier.</li>
<li class="my-0">🖼️ <strong>Image Optimization:</strong> Configures image handling for performance and security, enhancing user experience.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> JavaScript</li>
<li class="my-0"><strong>Package Manager:</strong> Npm</li>
</ul>
<h3>Installation</h3>
<p>Build traveloka-cloning from the source and intsall dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/adityagus/traveloka-cloning
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd traveloka-cloning
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">❯ npm install
</code></pre>
<h3>Usage</h3>
<p>Run the project with:</p>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">npm start
</code></pre>
